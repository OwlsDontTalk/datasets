# Fortune 1000

Structured version of the Fortune 1000 ranking for 2025 (for now),
publicly available list on https://us500.com.

Formats:
- CSV
- Parquet (pyarrow)

Fields:
- Rank
- Company
- Ticker
- Industry
- State
- City
- Revenue (millions USD)
- Profits (millions USD)
- Assets (millions USD)
- Market Value (millions USD)
- Employees
